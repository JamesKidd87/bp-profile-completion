== Description ==

BuddyPress Profile Completion Plugin allow site admin to force site members to fill all required profile fields, avatar and profile cover.

= Features:- =

1. Site admin can force site members to fill all required profile fields before using other features.
2. Site admin can force site members to upload an avatar.
3. Site admin can force site members to upload a cover photo for their profile.
4. Plugin allows site admin to redirect user to his profile page if user have not filled all the required things.
5. Plugin allows site admin to customize message for required fields, profile photo and cover.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.
1. Download the zip file and extract
1. Upload `bp-profile-completion` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu.
1. Enjoy

== Frequently Asked Questions ==

= Does This plugin works without BuddyPress =
No, It needs you to have BuddyPress installed and activated. Please make sure that you have BuddyPress enabled to use this plugin.
